# SQL Practice Queries

## Project Overview

This project contains SQL practice queries created using PostgreSQL. It covers SQL concepts from basic to intermediate level with well-organized query files for learning and practice.

## Topics Covered

* Database Creation
* Database Operations
* Data Manipulation
* Basic SQL Queries
* Aggregate Functions
* SQL Joins
* Set Operators
* Subqueries and Common Table Expressions (CTE)
* Window Functions
* String and Date Functions
* Views

## Database Tables

* Departments
* Employees
* Customers
* Products
* Orders
* Sales

This project was created to strengthen SQL fundamentals through hands-on practice and build a strong SQL portfolio.
